## App Row 1: Guidelines

![App Row 1 Banner](/assets/images/content/onsite/desktop/mobile-app-row1-guidelines.jpg)
{: .example-container }

#### TITLE
Min. size 44px Bold, Uppercase, anti-alias Strong

#### HEADLINE
Min. size 78px Bold, Uppercase, anti-alias Strong

#### T&C/FINE PRINT (IF NECESSARY)
Min. size 34px Regular, Uppercase, anti-alias Strong


	* Don’t include unnecessary long descriptions in mobile banners
	* Don’t use serif fonts (Baskerville)
	* Use UPPERCASE for better legibility
	* Use BOLD WEIGHT to headlines pop
	* Use anti-aliasing STRONG
